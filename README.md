# the-footy-lab

**the-footy-lab** is a beginner-friendly guide to the Australian Football League (AFL), also known as Australian/Aussie rules, Australian/Aussie football, or just footy.

**Current version:** v0.1

## Purpose

This project makes AFL rules, stats, and terminology easier to understand for beginners, overseas fans, and regular AFL followers.

## Tech stack

- Astro
- TypeScript
- Tailwind CSS
- Markdown
- GitHub

## Local setup

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Build the website:

```bash
npm run build
```
